# Election Guide India 🇮🇳

An interactive educational platform designed to empower Indian citizens with knowledge about the electoral process.

## 🚀 Live Demo
[View the Live App](https://prompt-war-5183385783.us-central1.run.app)

## ✨ Features
- **Interactive Learning:** Simplified modules on how elections work in India.
- **EVM Simulator:** A realistic, interactive simulator of an Electronic Voting Machine.
- **AI Assistant:** Get instant answers to your questions about voting and the constitution.
- **Flashcards:** Test your knowledge with curated election-related terminology.
- **Quiz:** Interactive quizzes to track your learning progress.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript
- **Deployment:** Google Cloud Run (Dockerized)

## 🐳 Running with Docker
```bash
docker build -t election-guide .
docker run -p 8080:80 election-guide
```
